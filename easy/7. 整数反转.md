# 7. 整数反转

给你一个 `32` 位的有符号整数 `x` ，返回将 `x` 中的数字部分反转后的结果。

如果反转后整数超过 `32` 位的有符号整数的范围 `[−231,  231 − 1]` ，就返回 `0`。

假设环境不允许存储 `64` 位整数（有符号或无符号）。


**示例 1：**
```
输入：x = 123
输出：321
```

**示例 2：**
```
输入：x = -123
输出：-321
```

**示例 3：**
```
输入：x = 120
输出：21
```

**示例 4：**
```
输入：x = 0
输出：0
```

**写法一**

```js
var reverse = function (x) {
  let res = 0;
  while (x) {
    res = res * 10 + x % 10;
    if (res < Math.pow(-2, 31) || res > Math.pow(2, 31) - 1) return 0;
    x = (x / 10) | 0; // 通过 | 0 取整，无论正负，只移除小数点部分（正数向下取整，负数向上取整）。
  }
  return res;
};
```
**写法二**

```js
var reverse = function(x) {
  let res = 0;
  while(x !== 0) {
      res = res * 10 + x % 10;
      x = (x / 10) | 0;
  }
  return (res | 0) === res ? res : 0; // res | 0 超过32位的整数转换结果不等于自身，可用作溢出判断。
};
```
